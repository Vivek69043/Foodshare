# FoodShare 🍱

A Flask-based web application to reduce food waste by connecting donors with NGOs and volunteers.

## Features
- Donate food
- Claim food
- Real-time updates
- Image upload support

## Tech Stack
- Python (Flask)
- HTML, CSS, JS
- JSON (data storage)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py
3. Open browser at:
   http://127.0.0.1:5000/
